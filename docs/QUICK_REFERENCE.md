## Adding Existing Content

Click **+ Add** to import content you've already written:
- Select type → Enter title → Paste content → Set status → Save
- Great for importing blog posts, published LinkedIn posts, etc.

---

## Generation Options

**From Sources:**
- Select 3-5 crawled articles or personal inputs
- Choose format(s)
- Add optional instructions

**From Existing Assets:**
- Select existing content (e.g., LinkedIn posts)
- Choose new format (e.g., Newsletter)
- Great for repurposing

**Prompt-Only:**
- No sources needed
- Just provide detailed instructions
- Uses your content profile

**Newsletters get:**
- 3 subject line options
- Preview text

## Common Workflows

### Weekly LinkedIn (5 posts)
1. Review week's crawled articles
2. Add personal notes
3. Generate 5 posts from best sources
4. Edit and schedule

### Monthly Newsletter
1. Select best 5-10 LinkedIn posts
2. Generate Newsletter format
3. Enrich for links + subject lines
4. Copy markdown, send via email platform

### Blog from Ideas
1. Add Manual Note with outline
2. Find 2-3 relevant articles
3. Generate Blog Post
4. Enrich for SEO + links
5. Copy to CMS

---

## Best Practices

| Do | Don't |
|----|-------|
| Use 3-5 sources per generation | Use too many sources |
| Add specific instructions | Leave instructions empty |
| Mix articles with personal notes | Rely only on external sources |
| Edit AI output before publishing | Publish without review |
| Keep content profile updated | Leave profile incomplete |

## Auto-Generation

Set it and forget it - wake up to fresh content drafts!

| Setting | Options |
|---------|---------|
| **Frequency** | Daily, Weekly (Sunday), Weekly (Monday) |
| **Time** | Any hour (e.g., 08:00 AM) |
| **Formats** | Select which content types to auto-generate |

**What happens:**
1. Gathers 10 most recent extractions (past 7 days)
2. Generates each selected format using your profile
3. Creates drafts for you to review
